# 視野マッパー

レーベル遺伝性視神経症（LHON）などで中心が見えづらい方向けに、
「中心からどのあたりで見えやすくなるか」を簡易的に測定・可視化するWebアプリです。

> ⚠️ 医療機器ではありません。結果は目安であり、診断には使えません。
> 正確な評価は眼科での視野検査を受けてください。

## 使い方

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

1. **画面の大きさ合わせ** — クレジットカード等を画面に当て、枠の幅を合わせる
   （画面のピクセル密度を較正します）
2. **距離の入力** — 目から画面までのおおよその距離を入力
3. **測定** — 白い画面の中央にある小さな赤い十字を見続けます。16方向に
   赤い点が最初から表示されており、1つずつ順番に外側へゆっくり動いて
   いきます。動いている点が「はっきり見えた」瞬間にスペースキーを押すか
   タップ。点はその場所にとどまります。これを16方向について行います（2〜3分）
4. **結果** — 方向ごとの「はっきり見えるようになった境界」をポーラーマップと表で表示

16方向すべての点を最初から表示しておくのは、刺激が突然現れると
反射的に視線がそちらへ動いてしまう（固視が外れる）のを防ぐためです。
応答済みの点をその場に残すのも同じ理由で、画面の見た目が急に変わらない
ようにしています。

設定画面の「結果画面のサンプルを見る」で、測定せずに結果のイメージを確認できます。

## 技術構成

- React 19 + TypeScript + Vite
- 依存ライブラリなし（描画はSVG）

主なファイル:

| ファイル | 内容 |
| --- | --- |
| `src/types.ts` | 型定義、方向の定数、視角→ピクセル変換 |
| `src/Setup.tsx` | キャリブレーションと説明の画面 |
| `src/Test.tsx` | 測定画面（赤い点を中心から外へ動かし、応答位置を記録） |
| `src/Result.tsx` | 結果画面（SVGポーラーマップと表） |
