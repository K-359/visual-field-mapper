# 視野マッパー

レーベル遺伝性視神経症（LHON）などで中心が見えづらい方向けに、
「視野内のどの場所で、どの大きさの刺激に気付けるか」を簡易的に
測定・可視化するWebアプリです。

> ⚠️ 医療機器ではありません。結果は目安であり、診断には使えません。
> 正確な評価は眼科での視野検査を受けてください。

## 使い方

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

1. **準備** — 画面から約20cmの距離をとり、測定する眼（右眼・左眼・両眼）を
   選びます。
   （距離は20cm固定、画面は一般的な96dpiディスプレイを想定しています）
2. **測定** — 画面中央の赤い十字を見続けます。3つの同心円上にそれぞれ
   共通する16方向、合計48箇所の固定地点があります。各地点を1回ずつ、
   全48試行をランダムな順番で表示し、円に気付いた時点でEnterを押します。
   Enterと同時に次の地点で拡大が始まります。Escでいつでも中止できます。
3. **結果** — 各地点で気付いた円の半径から、48箇所の値を滑らかに補間した
   グラデーションマップを表示します。濃い赤ほど、その位置で
   気付くまでに大きな刺激が必要だったことを示します。結果は自動的に
   測定履歴へ保存されます。

固視が外れにくいよう、円は測定中の1個だけを表示します。円は半径0から
1秒あたり視角0.15°の速さで、最大半径5°まで拡大します。最大サイズでも
画面から欠けない円形範囲に、3つの同心円を配置します。試行ごとに地点は
変わりますが、測定に使う48箇所そのものは毎回同じです。

設定画面の「結果のサンプル」で、測定せずに結果画面のイメージを確認できます。

## 測定履歴

測定が完了すると結果はブラウザのlocalStorageに保存されます（最大100件、
古いものから削除）。設定画面・結果画面の「測定履歴」から過去の結果の
表示・削除ができます。履歴は端末のブラウザ内にのみ保存されます。

## 技術構成

- React 19 + TypeScript + Vite
- 依存ライブラリなし（描画はSVG）

主なファイル:

| ファイル | 内容 |
| --- | --- |
| `src/types.ts` | 型定義、視角→ピクセル変換 |
| `src/App.tsx` | 画面遷移（設定 → 測定 → 結果 / 履歴） |
| `src/Setup.tsx` | 設定画面（眼の選択と説明） |
| `src/measurementPlan.ts` | 3つの同心円上の48地点と、48試行の順序生成 |
| `src/Test.tsx` | 測定画面（48個の固定地点をランダム順に表示） |
| `src/Result.tsx` | 結果画面（48地点の測定値をグラデーション表示） |
| `src/History.tsx` | 測定履歴の一覧・削除画面 |
| `src/historyStore.ts` | 履歴のlocalStorage保存・読み込み |
