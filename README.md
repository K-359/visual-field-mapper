# 視野マッパー

レーベル遺伝性視神経症（LHON）などで中心が見えづらい方向けに、
「中心からどのあたりで見えやすくなるか」を簡易的に測定・可視化するWebアプリです。

> ⚠️ 医療機器ではありません。結果は目安であり、診断には使えません。
> 正確な評価は眼科での視野検査を受けてください。

## 使い方

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

1. **準備** — 画面から約20cmの距離をとり、測定する眼（右眼・左眼・両眼）を
   選びます。測らない方の眼は手で軽くおおいます。
   （距離は20cm固定、画面は一般的な96dpiディスプレイを想定しています）
2. **測定** — 画面中央の赤い十字を見続けたまま、表示された赤い円を
   矢印キー（↑・→で外側、↓・←で内側）で動かし、円がくっきり見えた位置で
   Enterを押します。これを16方向（22.5°刻み）について1方向ずつ行います。
   Escでいつでも中止できます。
3. **結果** — 方向ごとの「円がくっきり見えるようになった境界」を
   SVGのポーラーマップと表で表示します。グレーの領域が見えづらかった範囲です。
   結果は自動的に測定履歴へ保存されます。

固視が外れにくいよう、円は操作中の1個だけを表示します。
円の初期位置は中心から視角2°、移動はキー1回あたり0.2°刻みで、
画面内に収まる範囲（最大30°）まで動かせます。

設定画面の「結果のサンプル」で、測定せずに結果画面のイメージを確認できます。

## 測定履歴

測定が完了すると結果はブラウザのlocalStorageに保存されます（最大100件、
古いものから削除）。設定画面・結果画面の「測定履歴」から過去の結果の
表示・削除ができます。履歴は端末のブラウザ内にのみ保存されます。

## 技術構成

- React 19 + TypeScript + Vite
- 依存ライブラリなし（描画はSVG）

主なファイル:

| ファイル | 内容 |
| --- | --- |
| `src/types.ts` | 型定義、16方向の定数、視角→ピクセル変換 |
| `src/App.tsx` | 画面遷移（設定 → 測定 → 結果 / 履歴） |
| `src/Setup.tsx` | 設定画面（眼の選択と説明） |
| `src/Test.tsx` | 測定画面（赤い円を矢印キーで動かし、見えた位置を記録） |
| `src/Result.tsx` | 結果画面（SVGポーラーマップと表） |
| `src/History.tsx` | 測定履歴の一覧・削除画面 |
| `src/historyStore.ts` | 履歴のlocalStorage保存・読み込み |
